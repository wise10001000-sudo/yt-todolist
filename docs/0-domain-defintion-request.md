너는 최고의 비즈니스 도메인 전문가야. "yt-todolist"라는 이름의 사용자 인증과 함께 사용하는 todolist애플리케이션을 개발하래고해. 다음 내용을 바탕으로 적절한 서브에이전트를 활용해 도메인 정의서 초안을 작성해줘.

# 문제 정의 :

- 사용자마다 할일 목록을 관리할 수 있는 앱이 있으면 좋겠다.
- 할일의 만료 기간이 표현되어 할일 목록 관리할수 있으면 좋겠다.
- 국경일 같은 일정(할일)이 함께 관리되면 좋겠다.

# todolist 앱의 주요 기능 :

- 회원가입을 한다음 로그인한 사용자만 앱의 주요 기능을 이용할 수 있다.
- 로그인한 사용자는 자신만의 할일 목록을 조회할 수 있다.
- 할일 목록을 추가, 삭제, 수정이 가능하다
- 기존 할일의 일정(시작,만료시기)과 내용을 수정할 수 있다.
- 기존할일목록 삭제하면 휴지통으로 들어가고 다시 원복할수 있어야 한다
- 휴지통에 포함된 일정은 되돌릴 수 없는 삭제를 할 수 있다.
- 국경일 같은 공통의 할일은 삭제할 수 없다.
